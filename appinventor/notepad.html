Создание блокнота для заметок

Учебное пособие по созданию приложения для заметок на MIT App Inventor

1

Цель проекта

Мы создадим полноценное приложение для заметок, которое позволит:

  • Создавать новые заметки
  • Просматривать список всех заметок
  • Редактировать существующие заметки
  • Удалять ненужные заметки
  • Сохранять заметки в памяти телефона
  • Искать заметки по тексту

Хранение данных

Мы будем использовать TinyDB - базу данных MIT App Inventor для сохранения заметок между запусками приложения.

Настройка базы данных

Добавьте компонент TinyDB в ваш проект:

// Инициализация при запуске приложения когда Screen1.Initialize вызвать ЗагрузитьЗаметкиИзБД установить ListPicker1.Elements = списокЗаметок // Процедура загрузки заметок процедура ЗагрузитьЗаметкиИзБД установить списокЗаметок = TinyDB1.GetValue("notes") если не списокЗаметок тогда установить списокЗаметок = создать список конец конец // Процедура сохранения заметок процедура СохранитьЗаметкиВБД TinyDB1.StoreValue("notes", списокЗаметок) конец

Структура заметки:

  • id - уникальный идентификатор заметки
  • title - заголовок заметки
  • text - текст заметки
  • date - дата создания
  • time - время создания
  • important - важная ли заметка

// Дополнение к коду кнопки "Сохранить" иначе если режимРедактирования = "edit" тогда // Находим заметку по id для каждого элемент в списокЗаметок если получить значение элемент ключ "id" = currentNoteId тогда установить в словарь элемент ключ "title" = titleTextBox.Text установить в словарь элемент ключ "text" = contentTextBox.Text установить в словарь элемент ключ "date" = DatePicker1.Date установить в словарь элемент ключ "time" = TimePicker1.Time установить в словарь элемент ключ "important" = CheckBox1.Checked конец конец вызвать СохранитьЗаметкиВБД открыть другой экран Screen1 конец

Удаление заметок

Код для кнопки удаления:

// Когда нажата кнопка "Удалить" если selectedNoteIndex > 0 тогда задать вопрос "Удалить эту заметку?" с заголовком "Подтверждение" если результат = "Да" тогда удалить из списка списокЗаметок selectedNoteIndex вызвать СохранитьЗаметкиВБД // Обновляем список установить ListPicker1.Elements = списокЗаметок Notifier1.ShowMessageDialog("Заметка удалена", "Успешно", "OK") конец конец

Безопасное удаление

Всегда спрашивайте подтверждение перед удалением. Можно добавить корзину, где заметки хранятся 30 дней перед полным удалением.

Поиск заметок

Добавим функцию поиска по тексту заметок:

// Когда изменяется текст в поле поиска установить searchText = TextBoxSearch.Text если searchText ≠ "" тогда установить filteredNotes = создать список для каждого note в списокЗаметок установить title = получить значение note ключ "title" установить text = получить значение note ключ "text" если содержит(title, searchText) или содержит(text, searchText) тогда добавить в список filteredNotes note конец конец установить ListPicker1.Elements = filteredNotes иначе установить ListPicker1.Elements = списокЗаметок конец

Улучшенный поиск

Можно добавить поиск по дате, фильтрацию по важным заметкам, или регистронезависимый поиск с помощью блока toUpperCase().

9

Экспорт и импорт заметок

Добавим возможность экспорта заметок в файл:

// Добавляем компонент File // Когда нажата кнопка "Экспорт" установить exportText = "" для каждого note в списокЗаметок установить exportText = объединить(exportText, "Заголовок: ") установить exportText = объединить(exportText, получить значение note ключ "title") установить exportText = объединить(exportText, "\n") установить exportText = объединить(exportText, "Текст: ") установить exportText = объединить(exportText, получить значение note ключ "text") установить exportText = объединить(exportText, "\n---\n") конец File1.SaveFile(exportText, "notes.txt") Notifier1.ShowMessageDialog("Заметки экспортированы в notes.txt", "Экспорт завершен", "OK")

Дополнительные задания

Попробуйте улучшить приложение блокнота:

  1. Категории заметок: Добавьте возможность группировать заметки по категориям (работа, учеба, личное)
  2. Напоминания: Используйте компонент Clock для создания напоминаний о заметках
  3. Синхронизация с облаком: Добавьте сохранение заметок в Google Drive или Dropbox
  4. Фотографии в заметках: Используйте компонент Camera для добавления фотографий к заметкам
  5. Резервное копирование: Создайте автоматическое резервное копирование раз в неделю
  6. Парольная защита: Добавьте пароль или отпечаток пальца для доступа к приложению
