Создание калькулятора

Учебное пособие по созданию калькулятора на MIT App Inventor

1

Подготовка рабочей среды

Для начала работы с MIT App Inventor необходимо:

  • Зарегистрироваться на сайте appinventor.mit.edu
  • Создать новый проект "Calculator"
  • Установить приложение MIT App Inventor Companion на свой Android-смартфон для тестирования

Совет

Если у вас нет Android-устройства, вы можете использовать эмулятор на компьютере. В меню "Подключить" выберите "Эмулятор".

Установка MIT App Inventor Companion

Скачайте приложение из Google Play или отсканируйте QR-код в MIT App Inventor. Оно нужно для тестирования приложений на телефоне в реальном времени.

2

Как расположить кнопки:

  1. TextBox разместите вверху экрана, сделайте его достаточно большим
  2. Кнопки расположите в виде сетки 4×5 как на обычном калькуляторе
  3. Первая строка: 7 8 9 ÷
  4. Вторая строка: 4 5 6 ×
  5. Третья строка: 1 2 3 -
  6. Четвертая строка: 0 . C +
  7. Пятая строка: = (на всю ширину)

Дизайнерские советы

Измените цвета кнопок: цифры - серый, операции - оранжевый, равно - зеленый, очистка - красный. Это сделает калькулятор более понятным.

3

Логика работы калькулятора

Перейдите во вкладку "Blocks" (Блоки). Нам понадобятся следующие глобальные переменные:

// Глобальные переменные для калькулятора firstNumber = "" // Первое число secondNumber = "" // Второе число operation = "" // Текущая операция (+, -, ×, ÷) isNewNumber = true // Флаг: ввод нового числа? currentResult = "" // Текущий результат

Принцип работы калькулятора:

  1. Пользователь нажимает цифры - они добавляются к текущему числу
  2. При нажатии на операцию (+, -, ×, ÷) сохраняем первое число и тип операции
  3. Пользователь вводит второе число
  4. При нажатии "=" выполняем вычисление и выводим результат
  5. Кнопка "C" очищает все данные и дисплей

Понимание логики

Переменная isNewNumber помогает понять, вводим ли мы новое число (после операции или очистки) или продолжаем ввод текущего числа.

4

Программирование кнопок цифр

Для каждой кнопки с цифрой создаем блок. Например, для кнопки "1":

// Когда нажата кнопка 1 если isNewNumber = true тогда установить DisplayTextBox.Text = "1" установить isNewNumber = false иначе установить DisplayTextBox.Text = объединить(DisplayTextBox.Text, "1") конец

Что делает этот код:

  • Если мы только начали вводить новое число (isNewNumber = true), то заменяем весь текст на "1"
  • Если мы продолжаем вводить число (isNewNumber = false), то добавляем "1" к текущему тексту
  • Устанавливаем isNewNumber = false, потому что теперь мы вводим число

Создайте аналогичные блоки для всех цифр от 0 до 9.

Быстрое копирование

Создайте блок для одной цифры, затем копируйте его (правый клик → Duplicate) и меняйте только цифру в блоке "установить Text".

5

Программирование кнопки точки

Кнопка точки (.) добавляет десятичную часть числа:

// Когда нажата кнопка . если не содержит(DisplayTextBox.Text, ".") тогда установить DisplayTextBox.Text = объединить(DisplayTextBox.Text, ".") установить isNewNumber = false конец

Особенности:

  • Проверяем, нет ли уже точки в числе (чтобы не было "5.2.1")
  • Если точки нет - добавляем ее
  • Если ввод был начат заново (isNewNumber = true), то нужно добавить "0." вместо просто "."

Улучшенная версия

Лучше сделать так:

если isNewNumber = true тогда установить DisplayTextBox.Text = "0." установить isNewNumber = false иначе если не содержит(DisplayTextBox.Text, ".") тогда установить DisplayTextBox.Text = объединить(DisplayTextBox.Text, ".") конец конец
6

Программирование операций

Для кнопки "+" создаем блок:

// Когда нажата кнопка + установить firstNumber = DisplayTextBox.Text установить operation = "+" установить isNewNumber = true

Аналогичные блоки создаем для операций -, ×, ÷.

Что происходит:

  1. Сохраняем текущее число из дисплея в переменную firstNumber
  2. Сохраняем тип операции в переменную operation
  3. Устанавливаем isNewNumber = true, чтобы следующая цифра начала новое число

Важно!

Для операции деления нужно добавить проверку деления на ноль перед выполнением вычисления. Мы сделаем это в блоке кнопки "=".

7

Программирование кнопки "="

Кнопка "равно" выполняет вычисление. Это самый сложный блок:

// Когда нажата кнопка = установить secondNumber = DisplayTextBox.Text если operation = "+" тогда установить currentResult = преобразовать_в_число(firstNumber) + преобразовать_в_число(secondNumber) иначе если operation = "-" тогда установить currentResult = преобразовать_в_число(firstNumber) - преобразовать_в_число(secondNumber) иначе если operation = "×" тогда установить currentResult = преобразовать_в_число(firstNumber) * преобразовать_в_число(secondNumber) иначе если operation = "÷" тогда если secondNumber = "0" тогда установить DisplayTextBox.Text = "Ошибка: деление на 0" вызвать ОчиститьВсе иначе установить currentResult = преобразовать_в_число(firstNumber) / преобразовать_в_число(secondNumber) конец конец // Выводим результат (если не было ошибки) если operation ≠ "" тогда установить DisplayTextBox.Text = currentResult установить firstNumber = currentResult установить isNewNumber = true установить operation = "" конец

Пояснения:

  • преобразовать_в_число() - функция, которая превращает текст в число
  • Проверка деления на ноль важна, иначе программа упадет с ошибкой
  • После вычисления результат становится новым firstNumber, чтобы можно было продолжать вычисления
8

Программирование кнопки "C"

Кнопка "C" (Clear) очищает все данные:

// Когда нажата кнопка C установить DisplayTextBox.Text = "0" установить firstNumber = "" установить secondNumber = "" установить operation = "" установить isNewNumber = true установить currentResult = ""

Также стоит создать процедуру (функцию) для очистки, чтобы использовать ее в разных местах:

// Процедура ОчиститьВсе установить DisplayTextBox.Text = "0" установить firstNumber = "" установить secondNumber = "" установить operation = "" установить isNewNumber = true установить currentResult = ""

Теперь в блоке кнопки "C" и в блоке деления на ноль можно просто вызывать ОчиститьВсе.

9

Тестирование и улучшения

После создания основных функций протестируйте калькулятор:

  • Проверьте простые операции: 5 + 3 = 8
  • Проверьте цепочку вычислений: 2 + 3 = 5 × 2 = 10
  • Проверьте деление на ноль: 5 ÷ 0 → ошибка
  • Проверьте работу с десятичными числами: 1.5 + 2.3 = 3.8
  • Проверьте кнопку очистки

Дополнительные функции

Попробуйте добавить эти улучшения:

  1. Кнопка "+/-" для изменения знака числа
  2. Кнопка "%" для вычисления процентов
  3. Кнопка "√" для квадратного корня
  4. История вычислений (показывает последние операции)
  5. Звуки при нажатии кнопок
  6. Темная/светлая тема

Профессиональный уровень

Создайте научный калькулятор с дополнительными функциями: sin, cos, tan, log, возведение в степень (x^y), факториал (n!).

10

Экспорт приложения

Готовое приложение можно установить на телефон:

  1. В меню "Build" выберите "App (save .apk to my computer)"
  2. Скачайте файл .apk на компьютер
  3. Перекиньте файл на телефон (по USB или через облако)
  4. На телефоне разрешите установку из неизвестных источников
  5. Установите приложение и запустите его!

Поделитесь приложением

Отправьте файл .apk друзьям и родственникам, чтобы они могли пользоваться вашим калькулятором!

Быстрая установка

В меню "Build" есть опция "Show QR Code". Отсканируйте QR-код с телефона, чтобы скачать приложение напрямую.
