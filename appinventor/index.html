MIT App Inventor

Создавай мобильные приложения без сложного кода!

Что такое MIT App Inventor?

MIT App Inventor - это визуальная среда программирования для создания мобильных приложений на Android. Не нужно писать код - просто перетаскивай компоненты и собирай приложение как конструктор!

Визуальное программирование

Собирай приложения из готовых блоков, как пазл

Для Android

Создавай приложения, которые работают на смартфонах и планшетах

Идеально для обучения

Отлично подходит для первого знакомства с созданием мобильных приложений

Работает в браузере

Не нужно ничего устанавливать - работай прямо в веб-браузере
