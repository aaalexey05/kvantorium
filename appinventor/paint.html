Создание рисовалки (Paint)

Учебное пособие по созданию графического редактора на MIT App Inventor

1

Основная идея рисовалки

Мы создадим приложение для рисования пальцем на экране смартфона с такими функциями:

  • Рисование линий пальцем
  • Выбор цвета из палитры
  • Регулировка толщины кисти
  • Ластик для стирания
  • Очистка всего холста
  • Сохранение рисунка в галерею

Как это работает

Рисовалка отслеживает движения пальца по Canvas (холст) и рисует линии между предыдущей и текущей позициями.

Глобальные переменные

Нам понадобятся переменные для хранения состояния рисовалки:

// Глобальные переменные текущийЦвет = черный // Текущий цвет рисования толщинаКисти = 5 // Текущая толщина линии режим = "рисование" // Режим: рисование или ластик prevX = 0 // Предыдущая координата X prevY = 0 // Предыдущая координата Y

Инициализация при запуске:

// Когда экран загружается установить текущийЦвет = сделать цвет (0, 0, 0) // Черный установить толщинаКисти = 5 установить режим = "рисование" установить Slider1.ThumbPosition = толщинаКисти установить LabelThickness.Text = объединить("Толщина: ", толщинаКисти)
4

Рисование на Canvas

Основная логика рисования при движении пальца:

// Когда палец коснулся Canvas (начало рисования) установить prevX = текущийX установить prevY = текущийY // Когда палец двигается по Canvas Canvas1.Dragged(текущийX, текущийY, prevX, prevY) если режим = "рисование" тогда Canvas1.DrawLine(prevX, prevY, текущийX, текущийY, толщинаКисти, текущийЦвет) иначе если режим = "ластик" тогда // Рисуем белым цветом (стираем) Canvas1.DrawLine(prevX, prevY, текущийX, текущийY, толщинаКисти, белый) конец // Сохраняем текущие координаты как предыдущие установить prevX = текущийX установить prevY = текущийY

Как работает рисование

При каждом движении пальца рисуется короткая линия между предыдущей и текущей позицией. Чем быстрее движется палец, тем длиннее эти отрезки.

Выбор цвета

Код для кнопок выбора цвета:

// Когда нажата кнопка красного цвета установить текущийЦвет = красный установить режим = "рисование" установить ButtonEraser.BackgroundColor = серый // Сбрасываем выделение ластика // Когда нажата кнопка синего цвета установить текущийЦвет = синий установить режим = "рисование" установить ButtonEraser.BackgroundColor = серый // Когда нажата кнопка зеленого цвета установить текущийЦвет = зеленый установить режим = "рисование" установить ButtonEraser.BackgroundColor = серый // Когда нажата кнопка черного цвета установить текущийЦвет = черный установить режим = "рисование" установить ButtonEraser.BackgroundColor = серый

Создание собственных цветов:

// Создание произвольного цвета RGB установить фиолетовый = сделать цвет (128, 0, 128) установить оранжевый = сделать цвет (255, 165, 0) установить розовый = сделать цвет (255, 192, 203) // Или использование Color Picker // Добавьте компонент ColorPickerButton когда ColorPickerButton.AfterPicking установить текущийЦвет = ColorPickerButton.SelectedColor установить режим = "рисование"

Ластик и очистка

Код для ластика и очистки холста:

// Когда нажата кнопка "Ластик" установить режим = "ластик" установить ButtonEraser.BackgroundColor = желтый // Подсвечиваем активный инструмент // Сбрасываем подсветку кнопок цветов установить ButtonRed.BackgroundColor = красный установить ButtonBlue.BackgroundColor = синий установить ButtonGreen.BackgroundColor = зеленый установить ButtonBlack.BackgroundColor = черный // Когда нажата кнопка "Очистить" задать вопрос "Очистить весь холст?" с заголовком "Подтверждение" если результат = "Да" тогда Canvas1.Clear() Notifier1.ShowMessageDialog("Холст очищен", "Готово", "OK") конец

Умный ластик

Можно сделать ластик, который стирает только определенный цвет или имеет разную форму (круг, квадрат, кисть).

Регулировка толщины кисти

Используем Slider для регулировки толщины:

// Когда меняется положение Slider установить толщинаКисти = Slider1.ThumbPosition установить LabelThickness.Text = объединить("Толщина: ", толщинаКисти) // Дополнительно: меняем размер индикатора толщины Canvas1.DrawCircle(50, 50, толщинаКисти / 2, текущийЦвет) Canvas1.Clear() // Очищаем индикатор

Предустановки толщины:

// Кнопка "Тонкая кисть" установить толщинаКисти = 2 установить Slider1.ThumbPosition = 2 // Кнопка "Средняя кисть" установить толщинаКисти = 5 установить Slider1.ThumbPosition = 5 // Кнопка "Толстая кисть" установить толщинаКисти = 10 установить Slider1.ThumbPosition = 10

Дополнительные инструменты

Добавим дополнительные инструменты для рисования:

// Рисование прямых линий процедура НарисоватьЛинию(x1, y1, x2, y2) Canvas1.DrawLine(x1, y1, x2, y2, толщинаКисти, текущийЦвет) конец // Рисование прямоугольников процедура НарисоватьПрямоугольник(x, y, ширина, высота) Canvas1.DrawRectangle(x, y, ширина, высота, ложь, толщинаКисти, текущийЦвет) конец // Рисование закрашенных прямоугольников процедура НарисоватьЗакрашенныйПрямоугольник(x, y, ширина, высота) Canvas1.DrawRectangle(x, y, ширина, высота, истина, 0, текущийЦвет) конец // Рисование кругов процедура НарисоватьКруг(x, y, радиус) Canvas1.DrawCircle(x, y, радиус, ложь, толщинаКисти, текущийЦвет) конец // Рисование текста процедура НарисоватьТекст(x, y, текст) Canvas1.DrawText(x, y, текст, текущийЦвет, 20) // 20 - размер шрифта конец

Творческие задания

Попробуйте добавить эти функции в свою рисовалку:

  1. Разные формы кистей: Сделайте кисти разной формы (круг, квадрат, звезда, сердечко)
  2. Градиентные цвета: Добавьте плавный переход цвета при рисовании
  3. Слои: Создайте систему слоев как в Photoshop
  4. Фильтры: Добавьте фильтры для готовых рисунков (черно-белый, сепия, инверсия)
  5. Анимация: Создайте простой анимационный редактор с кадрами
  6. Рисование по шаблону: Добавьте трафареты для рисования геометрических фигур
  7. Синхронизация: Сделайте совместное рисование на нескольких устройствах
