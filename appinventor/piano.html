Создание виртуального пианино

Учебное пособие по созданию музыкального приложения на MIT App Inventor

1

Идея проекта

Мы создадим виртуальное пианино с такими возможностями:

  • 8 октав пианино (белые и черные клавиши)
  • Реалистичное звучание нот
  • Запись и воспроизведение мелодий
  • Выбор разных музыкальных инструментов
  • Метроном для обучения ритму
  • Возможность сохранения композиций

Музыкальная теория

Пианино имеет 88 клавиш (52 белых, 36 черных). Мы создадим упрощенную версию на 24 клавиши (2 октавы).

2

Подготовка звуковых файлов

Нам понадобятся звуки для каждой ноты. Есть несколько способов:

  1. Скачать готовые сэмплы: Найти в интернете звуки пианино (формат .mp3 или .wav)
  2. Сгенерировать ноты: Использовать онлайн-генератор пианино
  3. Использовать встроенные звуки: MIT App Inventor имеет базовые звуки

Названия файлов:

C4.mp3 - нота До первой октавы D4.mp3 - нота Ре E4.mp3 - нота Ми F4.mp3 - нота Фа G4.mp3 - нота Соль A4.mp3 - нота Ля B4.mp3 - нота Си C5.mp3 - нота До второй октавы

Быстрый способ

Можно использовать компонент Sound и свойство Source - "BEEP", но это будет не реалистичное пианино.

3

Создание клавиш пианино

Создаем белые и черные клавиши с помощью кнопок:

Компоненты клавиш:

Button
× 14
Белые клавиши (широкие, светлые)
Button
× 10
Черные клавиши (узкие, темные, поверх белых)
HorizontalArrangement
→
Контейнер для одной октавы

Расположение клавиш:

  1. Белые клавиши: ширина 10%, высота 150px, цвет белый
  2. Черные клавиши: ширина 6%, высота 100px, цвет черный
  3. Расположите черные клавиши поверх белых с отступами
  4. Порядок клавиш: C, C#, D, D#, E, F, F#, G, G#, A, A#, B
4

Программирование звучания нот

Для каждой клавиши создаем отдельный компонент Sound:

// Пример для клавиши C (До) // В свойствах ButtonC установите: // Text = "C" // BackgroundColor = белый // Width = 10% // Height = 150px // Код для нажатия клавиши C когда ButtonC.Click SoundC.Play() установить ButtonC.BackgroundColor = светло-серый // Эффект нажатия если режимЗаписи = истина тогда добавить в список записанныеНоты "C4" добавить в список записанныеВремена Clock1.Now конец // Код для отпускания клавиши C когда ButtonC.TouchUp установить ButtonC.BackgroundColor = белый

Создайте аналогичные блоки для всех клавиш.

5

Запись и воспроизведение

Добавим возможность записи и воспроизведения мелодий:

// Глобальные переменные режимЗаписи = ложь записанныеНоты = список записанныеВремена = список началоЗаписи = 0 // Кнопка "Запись" когда ButtonRecord.Click если режимЗаписи = ложь тогда установить режимЗаписи = истина установить началоЗаписи = Clock1.Now установить записанныеНоты = создать список установить записанныеВремена = создать список установить ButtonRecord.Text = "⏹ Стоп" Notifier1.ShowMessageDialog("Запись началась", "Режим записи", "OK") иначе установить режимЗаписи = ложь установить ButtonRecord.Text = "⏺ Запись" Notifier1.ShowMessageDialog("Запись остановлена", "Готово", "OK") конец // Кнопка "Воспроизведение" когда ButtonPlay.Click если длина(записанныеНоты) > 0 тогда для i от 1 до длина(записанныеНоты) установить нота = выбрать элемент списка записанныеНоты i установить время = выбрать элемент списка записанныеВремена i если i = 1 тогда ждать 0 секунд иначе установить предыдущееВремя = выбрать элемент списка записанныеВремена (i - 1) установить пауза = время - предыдущееВремя ждать пауза секунд конец // Воспроизводим ноту если нота = "C4" тогда SoundC.Play() если нота = "D4" тогда SoundD.Play() // ... и так для всех нот конец иначе Notifier1.ShowMessageDialog("Нет записанной мелодии", "Ошибка", "OK") конец

Метроном

Добавим метроном для тренировки ритма:

// Глобальные переменные для метронома метрономАктивен = ложь темпМетронома = 60 // ударов в минуту счетчикМетронома = 0 // Компонент Clock для метронома когда ClockMetronome.Timer если метрономАктивен = истина тогда SoundMetronome.Play() установить счетчикМетронома = счетчикМетронома + 1 // Каждый 4-й удар громче если счетчикМетронома % 4 = 1 тогда установить SoundMetronome.Volume = 100 иначе установить SoundMetronome.Volume = 50 конец конец // Кнопка включения/выключения метронома когда ButtonMetronome.Click если метрономАктивен = ложь тогда установить метрономАктивен = истина установить счетчикМетронома = 0 установить ButtonMetronome.Text = "Метроном ⏸" // Настраиваем интервал таймера установить интервал = 60 / темпМетронома установить ClockMetronome.TimerInterval = интервал * 1000 // в миллисекундах ClockMetronome.TimerEnabled = истина иначе установить метрономАктивен = ложь установить ButtonMetronome.Text = "Метроном ▶" ClockMetronome.TimerEnabled = ложь конец // Slider для регулировки темпа когда SliderTempo.PositionChanged установить темпМетронома = SliderTempo.ThumbPosition установить LabelTempo.Text = объединить("Темп: ", темпМетронома, " BPM") если метрономАктивен = истина тогда установить интервал = 60 / темпМетронома установить ClockMetronome.TimerInterval = интервал * 1000 конец

Добавим возможность сохранения созданных мелодий:

// Сохранение мелодии процедура СохранитьМелодию(название) если длина(записанныеНоты) > 0 тогда установить data = создать список для i от 1 до длина(записанныеНоты) установить нота = выбрать элемент списка записанныеНоты i установить время = выбрать элемент списка записанныеВремена i установить элементЗаписи = создать словарь установить в словарь элементЗаписи ключ "note" = нота установить в словарь элементЗаписи ключ "time" = время добавить в список data элементЗаписи конец TinyDB1.StoreValue(название, data) Notifier1.ShowMessageDialog(объединить("Мелодия '", название, "' сохранена"), "Успешно", "OK") иначе Notifier1.ShowMessageDialog("Нет записанной мелодии для сохранения", "Ошибка", "OK") конец конец // Загрузка мелодии процедура ЗагрузитьМелодию(название) установить data = TinyDB1.GetValue(название) если data тогда установить записанныеНоты = создать список установить записанныеВремена = создать список для каждого элемент в data установить нота = получить значение элемент ключ "note" установить время = получить значение элемент ключ "time" добавить в список записанныеНоты нота добавить в список записанныеВремена время конец Notifier1.ShowMessageDialog(объединить("Мелодия '", название, "' загружена"), "Готово", "OK") иначе Notifier1.ShowMessageDialog(объединить("Мелодия '", название, "' не найдена"), "Ошибка", "OK") конец конец

Дополнительные задания

Попробуйте добавить эти функции в ваше пианино:

  1. Аккорды: Добавьте кнопки для автоматического воспроизведения аккордов (C-мажор, G-мажор и т.д.)
  2. Эффекты: Добавьте реверберацию, эхо, дисторшн для звука
  3. Обучение: Создайте режим обучения с подсветкой клавиш для известных мелодий
  4. Нотный стан: Добавьте визуализацию нот на нотном стане в реальном времени
  5. Мультитач: Сделайте поддержку нескольких одновременных нажатий (аккорды)
  6. Синтезатор: Добавьте возможности синтезатора с генерацией собственных звуков
  7. Барабанная установка: Добавьте панель с ударными инструментами
