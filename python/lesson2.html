Урок 2: Переменные и типы данных

Учимся хранить информацию в программе

Что такое переменная?

Переменная — это как коробка с названием, в которую можно положить какое-то значение (число, текст и т.д.).

# Создаем переменную и сохраняем в нее значение
name = "Анна"
age = 12
height = 1.45
is_student = True

В этом примере:

  • name - переменная для хранения имени (текст)
  • age - переменная для хранения возраста (целое число)
  • height - переменная для хранения роста (дробное число)
  • is_student - переменная для хранения информации, учится ли человек (логическое значение)

Основные типы данных в Python

В Python есть несколько основных типов данных:

Тип данных Пример Описание
int (целые числа) 10, -5, 0, 1000 Целые числа без дробной части
float (дробные числа) 3.14, -2.5, 0.0 Числа с плавающей точкой (дробные)
str (строки) "Привет", 'мир', "123" Текст, обрамленный кавычками
bool (логические) True, False Логические значения "истина" или "ложь" 
# Примеры разных типов данных
number = 42  # целое число (int)
pi = 3.14159  # дробное число (float)
greeting = "Hello, World!"  # строка (str)
is_raining = False  # логическое значение (bool)

# Можно узнать тип любой переменной с помощью type()
print(type(number))  # Выведет: 
print(type(pi))      # Выведет: 
print(type(greeting))# Выведет: 
print(type(is_raining))# Выведет:

Работа со строками

Строки (текст) — один из самых часто используемых типов данных. Вот что можно делать со строками:

# Создание строк
name = "Мария"
city = 'Москва'
message = "Я изучаю Python!"

# Конкатенация (сложение) строк
full_name = name + " Иванова"
print(full_name)  # Выведет: Мария Иванова

# Умножение строк
lots_of_hello = "Hello! " * 3
print(lots_of_hello)  # Выведет: Hello! Hello! Hello! 

# Длина строки
word = "программирование"
length = len(word)
print(f"Длина слова '{word}' = {length} символов")

Ввод данных от пользователя

Часто программы получают данные от пользователя. Для этого используется функция input():

# Простой ввод данных
name = input("Как тебя зовут? ")
print(f"Привет, {name}! Рад познакомиться!")

# Ввод чисел
age = input("Сколько тебе лет? ")
# input() всегда возвращает строку, поэтому преобразуем в число
age = int(age)
print(f"Через 10 лет тебе будет {age + 10} лет")

Важно!

Функция input() всегда возвращает строку, даже если пользователь ввел число. Чтобы работать с числом, нужно преобразовать строку в число с помощью int() (для целых чисел) или float() (для дробных).

Форматирование строк (f-строки)

В Python 3.6+ появились удобные f-строки для форматирования:

name = "Кирилл"
age = 14
height = 1.65

# Старый способ (не рекомендуется)
print("Меня зовут %s, мне %d лет, мой рост %.2f м" % (name, age, height))

# Современный способ (f-строки) - РЕКОМЕНДУЕТСЯ!
print(f"Меня зовут {name}, мне {age} лет, мой рост {height} м")

# В f-строках можно делать вычисления
print(f"Через 5 лет мне будет {age + 5} лет")

# Форматирование чисел
price = 123.456789
print(f"Цена: {price:.2f} руб.")  # Округление до 2 знаков после запятой

Практическое задание

Создайте программу "Анкета", которая:

  1. Спрашивает у пользователя его имя, возраст, город проживания и любимый предмет в школе
  2. Сохраняет все ответы в переменные
  3. Выводит красивую анкету с использованием f-строк
  4. Добавляет вычисление: сколько лет было пользователю 5 лет назад

Пример результата:

====== АНКЕТА ======
Имя: Анна
Возраст: 12 лет
Город: Барнаул
Любимый предмет: Математика
5 лет назад мне было 7 лет
===================

Подсказка: используйте input() для ввода, не забудьте преобразовать возраст в число с помощью int().

Полезные советы

  • Давайте переменным понятные имена: user_age вместо a, total_price вместо tp
  • Python чувствителен к регистру: Name и name - разные переменные
  • Используйте f-строки для форматирования вывода - это самый современный и удобный способ
  • Комментируйте свой код, чтобы помнить, что делает каждая переменная
