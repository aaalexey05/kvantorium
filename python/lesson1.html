Урок 1: Начало работы с Python

Установи Python и напиши свою первую программу!

Шаг 1: Установка Python

Для начала работы с Python нужно установить его на свой компьютер:

  1. Перейдите на официальный сайт python.org
  2. Скачайте последнюю версию Python (рекомендуем Python 3.x)
  3. Запустите установщик и следуйте инструкциям
  4. Обязательно отметьте галочку "Add Python to PATH"

Совет

После установки откройте командную строку (Terminal на Mac/Linux или Command Prompt на Windows) и введите python --version. Если вы видите версию Python, значит установка прошла успешно!

Шаг 2: Выбор среды разработки

Для написания программ на Python вам понадобится текстовый редактор или среда разработки (IDE). Вот несколько вариантов:

  • IDLE - простая среда, которая идет вместе с Python
  • Thonny - отличный выбор для начинающих
  • VS Code - мощный редактор с поддержкой Python
  • PyCharm - профессиональная среда разработки

Для начала мы рекомендуем использовать Thonny - он простой и понятный для новичков.

Шаг 3: Первая программа "Привет, мир!"

Традиционно первой программой на любом языке программирования является вывод фразы "Hello, World!" на экран. В Python это делается очень просто:

print("Привет, мир!")
print("Я изучаю Python!")
print("Это мой первый урок!")

Давайте разберем эту программу:

  • print() - это функция, которая выводит текст на экран
  • Текст, который мы хотим вывести, помещается в кавычки внутри скобок
  • Каждая новая команда пишется с новой строки

Шаг 4: Сохранение и запуск программы

Чтобы сохранить и запустить свою программу:

  1. Откройте выбранную вами среду разработки (например, Thonny)
  2. Напишите программу из предыдущего шага
  3. Сохраните файл с расширением .py (например, hello.py)
  4. Нажмите кнопку "Run" или "Запустить"

Если все сделано правильно, вы увидите результат выполнения программы в окне вывода.

Шаг 5: Работа с комментариями

Комментарии - это пояснения в коде, которые игнорируются Python. Они помогают понять, что делает программа.

# Это однострочный комментарий
# Он начинается с символа решетки (#)

print("Привет, мир!")  # А это комментарий в конце строки

"""
Это многострочный комментарий.
Он полезен для длинных пояснений.
Такие комментарии обрамляются тройными кавычками.
"""

Комментарии - важная часть программирования! Всегда пишите комментарии к своему коду.

Практическое задание

Создайте программу, которая выводит на экран:

  1. Ваше имя
  2. Ваш возраст
  3. Ваш любимый цвет
  4. Пустую строку между каждым пунктом

Пример результата:

Меня зовут Анна
Мне 12 лет

Мой любимый цвет - синий

Подсказка: используйте функцию print() и пустые строки можно выводить так: print() или print("").

Советы для успешного обучения

  • Не бойтесь ошибаться! Ошибки - это нормальная часть обучения
  • Практикуйтесь каждый день, даже по 15-20 минут
  • Сохраняйте все свои программы - они пригодятся в будущем
  • Задавайте вопросы, если что-то непонятно
