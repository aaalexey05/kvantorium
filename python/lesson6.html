Урок 6: Словари и множества

Работаем с неупорядоченными коллекциями данных

Что такое словари?

Словарь (dict) — это неупорядоченная коллекция пар "ключ-значение". Ключ должен быть уникальным и неизменяемым объектом.

"name"
→
"Анна"
"age"
→
12
"city"
→
"Москва"
# Создание словарей
student = {
    "name": "Анна",
    "age": 12,
    "city": "Москва",
    "grades": [5, 4, 5, 4, 5]
}

phonebook = {
    "Мама": "+7 999 123-45-67",
    "Папа": "+7 999 987-65-43",
    "Друг": "+7 999 111-22-33"
}

empty_dict = {}        # Пустой словарь
empty_dict2 = dict()   # Тоже пустой словарь

# Доступ к значениям
print(student["name"])      # Анна
print(phonebook["Мама"])    # +7 999 123-45-67

# Изменение значений
student["age"] = 13
student["school"] = "Школа №123"  # Добавление нового ключа

# Удаление
del student["city"]
removed = student.pop("grades")  # Удаляет и возвращает значение

Методы словарей

У словарей есть много полезных методов:

Метод Описание Пример
get(key, default) Безопасное получение значения student.get("name", "неизвестно")
keys() Все ключи student.keys()
values() Все значения student.values()
items() Пары ключ-значение student.items()
update() Объединение словарей d1.update(d2)
clear() Очистка словаря student.clear()
copy() Копия словаря new = student.copy()
setdefault() Получить или создать ключ student.setdefault("class", "7А") 
# Практическое использование методов
student = {"name": "Анна", "age": 12}

# Безопасный доступ
print(student.get("city", "Город не указан"))  # Город не указан

# Итерация по словарю
for key in student.keys():
    print(f"Ключ: {key}")

for value in student.values():
    print(f"Значение: {value}")

for key, value in student.items():
    print(f"{key}: {value}")

# Объединение словарей
info1 = {"name": "Анна"}
info2 = {"age": 12, "city": "Москва"}
info1.update(info2)
print(info1)  # {'name': 'Анна', 'age': 12, 'city': 'Москва'}

Вложенные словари

Словари могут содержать другие словари:

# Сложная структура данных
school = {
    "name": "Школа №123",
    "address": "ул. Пушкина, 10",
    "students": {
        "Анна": {
            "age": 12,
            "class": "7А",
            "grades": {"математика": 5, "русский": 4, "информатика": 5}
        },
        "Иван": {
            "age": 13,
            "class": "7Б",
            "grades": {"математика": 4, "русский": 4, "информатика": 5}
        }
    },
    "teachers": ["Иванова И.И.", "Петров П.П.", "Сидорова С.С."]
}

# Доступ к вложенным данным
print(school["students"]["Анна"]["grades"]["математика"])  # 5

# Добавление нового ученика
school["students"]["Мария"] = {
    "age": 12,
    "class": "7А",
    "grades": {"математика": 5, "русский": 5}
}

# Обновление оценок
school["students"]["Анна"]["grades"]["физика"] = 4

Генераторы словарей

Как и списки, словари можно создавать с помощью генераторов:

# Квадраты чисел
squares = {x: x**2 for x in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Фильтрация
numbers = {1: "один", 2: "два", 3: "три", 4: "четыре"}
even_numbers = {k: v for k, v in numbers.items() if k % 2 == 0}
print(even_numbers)  # {2: 'два', 4: 'четыре'}

# Преобразование списка в словарь
students = ["Анна", "Иван", "Мария"]
students_dict = {i+1: name for i, name in enumerate(students)}
print(students_dict)  # {1: 'Анна', 2: 'Иван', 3: 'Мария'}

# Объединение двух списков в словарь
keys = ["name", "age", "city"]
values = ["Анна", 12, "Москва"]
person = {k: v for k, v in zip(keys, values)}
print(person)  # {'name': 'Анна', 'age': 12, 'city': 'Москва'}

Множества (sets)

Множество — это неупорядоченная коллекция уникальных элементов:

# Создание множеств
fruits = {"яблоко", "банан", "апельсин", "яблоко"}  # Дубликаты удаляются
print(fruits)  # {'банан', 'апельсин', 'яблоко'} (порядок может быть другим)

numbers = set([1, 2, 3, 4, 5, 2, 3])  # Из списка
print(numbers)  # {1, 2, 3, 4, 5}

empty_set = set()  # Пустое множество (НЕ {} - это словарь!)

# Основные операции
fruits.add("киви")          # Добавить элемент
fruits.remove("банан")      # Удалить элемент (если нет - ошибка)
fruits.discard("груша")     # Удалить (если нет - без ошибки)
popped = fruits.pop()       # Удалить и вернуть случайный элемент
fruits.clear()              # Очистить множество

Операции над множествами

Множества поддерживают математические операции:

A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

# Объединение
print(A | B)        # {1, 2, 3, 4, 5, 6, 7, 8}
print(A.union(B))   # То же самое

# Пересечение
print(A & B)        # {4, 5}
print(A.intersection(B))

# Разность
print(A - B)        # {1, 2, 3} (элементы из A, которых нет в B)
print(B - A)        # {6, 7, 8} (элементы из B, которых нет в A)

# Симметричная разность
print(A ^ B)        # {1, 2, 3, 6, 7, 8} (элементы, которые есть только в одном из множеств)

# Проверка отношений
print(A <= B)       # False (A не является подмножеством B)
print({1, 2} <= A)  # True ({1, 2} является подмножеством A)
print(A >= B)       # False (A не является надмножеством B)

Замороженные множества (frozenset)

Неизменяемые множества:

# Создание frozenset
colors = frozenset(["красный", "зеленый", "синий"])
print(colors)  # frozenset({'красный', 'зеленый', 'синий'})

# Что можно делать
print("красный" in colors)  # True
print(len(colors))          # 3
print(colors & {"красный", "желтый"})  # frozenset({'красный'})

# Что НЕЛЬЗЯ делать
# colors.add("желтый")      # ОШИБКА!
# colors.remove("красный")  # ОШИБКА!

# Использование в качестве ключей словаря
# Обычные множества не могут быть ключами, а frozenset - могут
data = {
    frozenset([1, 2]): "пары",
    frozenset([3, 4, 5]): "тройки"
}
print(data[frozenset([1, 2])])  # 'пары'

Практическое задание: "Телефонная книга"

Создайте программу для управления телефонной книгой.

Требования:

  1. Храните контакты в словаре: имя → список телефонов
  2. Реализуйте функции:
    • Добавить контакт
    • Удалить контакт
    • Найти контакт
    • Показать все контакты
    • Добавить телефон к существующему контакту
    • Удалить телефон у контакта
    • Экспорт/импорт в файл
  3. Добавьте группировку контактов (работа, семья, друзья)
  4. Реализуйте поиск по части имени или номера
  5. Используйте множества для хранения групп

Пример структуры данных:

phonebook = {
    "Мама": {
        "phones": ["+7 999 111-11-11", "+7 999 222-22-22"],
        "groups": {"семья", "важные"}
    },
    "Иван": {
        "phones": ["+7 999 333-33-33"],
        "groups": {"друзья", "работа"}
    }
}

groups = {
    "семья": {"Мама", "Папа"},
    "друзья": {"Иван", "Мария", "Сергей"},
    "работа": {"Иван", "Начальник"}
}

Пример работы программы:

=== ТЕЛЕФОННАЯ КНИГА ===
1. Добавить контакт
2. Удалить контакт
3. Найти контакт
4. Показать все
5. Показать по группам
6. Экспорт в файл
7. Выход

Выберите действие: 4

Контакты:
Мама:
  - +7 999 111-11-11 (домашний)
  - +7 999 222-22-22 (рабочий)
  Группы: семья, важные

Иван:
  - +7 999 333-33-33 (мобильный)
  Группы: друзья, работа

Подсказки:

  • Используйте вложенные словари для хранения информации о контактах
  • Используйте множества для групп (быстрый поиск и операции)
  • Для работы с файлами используйте модуль json
  • Реализуйте поиск с помощью генераторов словарей

Полезные советы

  • Используйте словари для структурированных данных
  • Используйте множества для хранения уникальных значений
  • Метод get() безопаснее, чем прямой доступ по ключу
  • Генераторы словарей делают код чище
  • Используйте items() для итерации по словарю
  • Множества эффективны для операций с уникальными элементами
  • Используйте frozenset как ключи словаря
