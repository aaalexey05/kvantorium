Урок 4: Циклы

Повторяем действия автоматически

Что такое циклы?

Циклы позволяют выполнять один и тот же блок кода несколько раз. Это как сказать программе: "Сделай это 5 раз" или "Продолжай делать это, пока условие истинно".

Пример из жизни:

1
Представьте, что вам нужно поздороваться с 10 друзьями
2
Без цикла: print("Привет, друг 1!"), print("Привет, друг 2!"), ... (10 строк)
3
С циклом: for i in range(10): print(f"Привет, друг {i+1}!") (1 строка!)

В Python есть два основных типа циклов: for и while.

Цикл for

Цикл for используется, когда известно, сколько раз нужно повторить действие.

# Самый простой цикл for
for i in range(5):
    print(f"Итерация номер {i+1}")
    
# Результат:
# Итерация номер 1
# Итерация номер 2
# Итерация номер 3
# Итерация номер 4
# Итерация номер 5

Функция range() создает последовательность чисел:

Вызов Результат Пример
range(5) 0, 1, 2, 3, 4 for i in range(5):
range(2, 7) 2, 3, 4, 5, 6 for i in range(2, 7):
range(1, 10, 2) 1, 3, 5, 7, 9 for i in range(1, 10, 2):
range(10, 0, -1) 10, 9, 8, ..., 1 for i in range(10, 0, -1):

Запомните!

В Python отсчет всегда начинается с 0! range(5) даст числа 0, 1, 2, 3, 4, а не 1, 2, 3, 4, 5.

Цикл for с элементами списка

Цикл for может проходить по элементам любого набора данных, например, списка:

# Цикл по списку фруктов
fruits = ["яблоко", "банан", "апельсин", "киви"]

for fruit in fruits:
    print(f"Я люблю {fruit}!")

# Результат:
# Я люблю яблоко!
# Я люблю банан!
# Я люблю апельсин!
# Я люблю киви!

# Цикл по строкам
message = "Python"
for letter in message:
    print(letter)

# Результат:
# P
# y
# t
# h
# o
# n

Цикл while

Цикл while выполняется, пока условие истинно. Используется, когда неизвестно, сколько раз нужно повторить действие.

# Простой цикл while
count = 1
while count <= 5:
    print(f"Счетчик: {count}")
    count = count + 1  # Увеличиваем счетчик на 1

# Результат:
# Счетчик: 1
# Счетчик: 2
# Счетчик: 3
# Счетчик: 4
# Счетчик: 5

Осторожно с бесконечными циклами!

Если забыть изменить переменную в условии, цикл может стать бесконечным:

# НЕПРАВИЛЬНО! Бесконечный цикл!
count = 1
while count <= 5:
    print(f"Счетчик: {count}")
    # Забыли увеличить count - программа зависнет!

Операторы break и continue

Эти операторы позволяют управлять выполнением цикла:

# break - полностью выходит из цикла
print("Пример с break:")
for i in range(1, 11):
    if i == 5:
        print("Нашли 5, выходим из цикла!")
        break
    print(i)

# Результат: 1 2 3 4 "Нашли 5..."

# continue - пропускает текущую итерацию
print("\nПример с continue:")
for i in range(1, 11):
    if i % 2 == 0:  # Если число четное
        continue    # Пропускаем его
    print(i)        # Выводим только нечетные числа

# Результат: 1 3 5 7 9

Разница между break и continue:

  • break - полностью останавливает весь цикл
  • continue - пропускает только текущую итерацию, цикл продолжается

Вложенные циклы

Циклы можно вкладывать друг в друга:

# Таблица умножения с вложенными циклами
print("Таблица умножения:")

for i in range(1, 4):  # Внешний цикл: числа от 1 до 3
    for j in range(1, 4):  # Внутренний цикл: тоже от 1 до 3
        result = i * j
        print(f"{i} × {j} = {result}")
    print()  # Пустая строка между блоками

Как работают вложенные циклы:

1
Внешний цикл: i = 1
2
Внутренний цикл проходит все j от 1 до 3: 1×1=1, 1×2=2, 1×3=3
3
Внешний цикл: i = 2
4
Внутренний цикл проходит все j от 1 до 3: 2×1=2, 2×2=4, 2×3=6
5
И так далее...

Цикл for с else

В Python у цикла for есть необязательная часть else, которая выполняется после завершения цикла (но не после break):

# Поиск числа в списке
numbers = [1, 3, 5, 7, 9]
search_for = 5

for num in numbers:
    if num == search_for:
        print(f"Нашли число {search_for}!")
        break
else:
    # Выполнится только если break НЕ был вызван
    print(f"Число {search_for} не найдено в списке")

Блок else в цикле выполняется только если цикл завершился нормально (без использования break).

Практическое задание: "Игра Угадай число"

Создайте игру, где компьютер загадывает число от 1 до 100, а игрок должен его угадать.

Требования:

  1. Компьютер загадывает случайное число от 1 до 100 (используйте import random и random.randint(1, 100))
  2. Игрок вводит свои предположения
  3. Компьютер говорит "слишком мало", "слишком много" или "угадал!"
  4. Считайте количество попыток
  5. После угадывания покажите статистику игры
  6. Спросите, хочет ли игрок сыграть еще раз

Пример работы программы:

Я загадал число от 1 до 100. Угадай его!
Введи число: 50
Слишком много!
Введи число: 25
Слишком мало!
Введи число: 37
Слишком много!
Введи число: 31
Ура! Ты угадал число 31 за 4 попытки!

Хочешь сыграть еще? (да/нет): да

Подсказки:

  • Используйте цикл while True: для основной игры
  • Внутри него используйте еще один цикл для угадывания
  • Используйте break для выхода из внутреннего цикла при угадывании
  • Используйте if answer.lower() == "нет": break для выхода из внешнего цикла

Полезные советы

  • Используйте for, когда знаете количество повторений
  • Используйте while, когда повторения зависят от условия
  • Всегда проверяйте, чтобы в цикле while условие рано или поздно стало ложным
  • Используйте range() с тремя параметрами для создания сложных последовательностей
  • Не бойтесь использовать break и continue, они делают код чище
