Урок 3: Условные операторы

Учим программу принимать решения

Что такое условные операторы?

Условные операторы позволяют программе выполнять разные действия в зависимости от условий. Это как развилка на дороге: если выполняется определенное условие, программа идет по одному пути, если нет — по другому.

# Простой пример
age = 15

if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

В этом примере программа проверяет, больше или равно ли age 18. Если да, выводит первое сообщение, если нет — второе.

Оператор if

Основной оператор условия — if (если).

# Синтаксис оператора if
if условие:
    # код, который выполнится, если условие истинно
    действие1()
    действие2()
    # ... остальной код с отступом

# Код после блока if (без отступа) выполнится в любом случае
print("Программа продолжает работу")

Важно! Все действия внутри блока if должны иметь одинаковый отступ (обычно 4 пробела).

Обратите внимание на двоеточие!

После условия в операторе if всегда ставится двоеточие :. Это важный синтаксис Python, без него программа не будет работать.

Операторы сравнения

Для создания условий используются операторы сравнения:

Оператор Пример Описание
== a == b Равно (проверяет, равны ли значения)
!= a != b Не равно
< a < b Меньше
> a > b Больше
<= a <= b Меньше или равно
>= a >= b Больше или равно 
# Примеры использования операторов сравнения
temperature = 25

if temperature > 30:
    print("Жарко! Включи кондиционер")
elif temperature > 20:
    print("Тепло, можно гулять")
elif temperature > 10:
    print("Прохладно, оденься теплее")
else:
    print("Холодно, лучше останься дома")

Операторы else и elif

else (иначе) и elif (иначе если) используются вместе с if для создания сложных условий.

# Полная структура условий
score = 85

if score >= 90:
    grade = "Отлично"
    print("Пять! Молодец!")
elif score >= 70:
    grade = "Хорошо"
    print("Четыре! Неплохо!")
elif score >= 50:
    grade = "Удовлетворительно"
    print("Тройка. Могло быть и лучше.")
else:
    grade = "Неудовлетворительно"
    print("Двойка. Нужно больше заниматься.")

print(f"Твоя оценка: {grade}")

Программа проверяет условия сверху вниз. Как только найдет первое истинное условие, выполнит соответствующий блок кода и выйдет из всей цепочки условий.

Логические операторы

Для создания сложных условий используются логические операторы:

Оператор Пример Описание
and a > 0 and a < 10 И (истинно, если оба условия истинны)
or a == 0 or b == 0 ИЛИ (истинно, если хотя бы одно условие истинно)
not not a > 0 НЕ (инвертирует значение условия) 
# Примеры с логическими операторами
age = 16
has_ticket = True

# Проверка нескольких условий с помощью and
if age >= 12 and age <= 18:
    print("Ты подросток!")

# Проверка с помощью or
if age < 6 or age > 60:
    print("Тебе положена льгота")

# Проверка на разрешение посещения
if age >= 12 and has_ticket:
    print("Добро пожаловать на фильм!")
else:
    print("Извини, ты не можешь посмотреть этот фильм")

# Использование not
is_raining = False
if not is_raining:
    print("Дождь не идет, можно идти гулять")

Вложенные условия

Условия можно вкладывать друг в друга:

# Пример вложенных условий
age = 15
has_parent_permission = True

if age >= 18:
    print("Вы можете зарегистрироваться")
else:
    print("Вам меньше 18 лет")
    
    if age >= 16 and has_parent_permission:
        print("Но с разрешения родителей можно")
    elif age >= 14:
        print("Можно только просматривать контент")
    else:
        print("Извините, доступ запрещен")

Будьте осторожны с отступами!

При вложенных условиях важно правильно расставлять отступы. Каждый новый уровень вложенности добавляет 4 пробела отступа.

Практическое задание: "Калькулятор оценок"

Создайте программу, которая:

  1. Запрашивает у пользователя количество баллов (от 0 до 100)
  2. Определяет оценку по системе:
    • 90-100: "Отлично" (5)
    • 70-89: "Хорошо" (4)
    • 50-69: "Удовлетворительно" (3)
    • 0-49: "Неудовлетворительно" (2)
  3. Если пользователь ввел число вне диапазона 0-100, выводит сообщение об ошибке
  4. Для отличных оценок добавляет похвалу, для неудовлетворительных - совет подтянуть знания

Пример работы программы:

Введите количество баллов: 95
Ваша оценка: Отлично (5)
Поздравляем! Отличный результат!

Введите количество баллов: 45
Ваша оценка: Неудовлетворительно (2)
Не расстраивайся! Больше практикуйся!

Введите количество баллов: 105
Ошибка! Баллы должны быть от 0 до 100

Подсказка: используйте цепочку if-elif-else и проверку на корректность ввода с помощью оператора and.

Полезные советы

  • Всегда проверяйте граничные случаи (например, что будет при score = 70)
  • Используйте elif вместо нескольких if, если условия взаимоисключающие
  • Пишите понятные условия: if age >= 18: вместо if not age < 18:
  • Не бойтесь использовать вложенные условия, но старайтесь не делать слишком глубокую вложенность
