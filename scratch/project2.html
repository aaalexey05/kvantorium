Проект 2: Игра "Поймай яблоко"

Создай свою первую игру с подсчетом очков и уровнями сложности

1

Идея игры

Мы создадим игру, где игрок управляет корзинкой и должен ловить падающие яблоки. За каждое пойманное яблоко начисляются очки.

Что нужно для игры:

  • Корзинка (управляется стрелками влево/вправо)
  • Падающие яблоки (появляются в случайных местах)
  • Счетчик очков
  • Таймер или счетчик жизней

Механика игры

Игрок двигает корзинку стрелками. Яблоки падают сверху. Если яблоко касается корзинки - игрок получает очко. Если яблоко падает мимо - теряет жизнь.

2

Подготовка спрайтов

Создайте новый проект в Scratch. Удалите котика (правый клик → удалить).

Добавьте спрайты:

  1. Корзинка: нажмите "Выбрать спрайт" → введите "basket" в поиске
  2. Яблоко: "Выбрать спрайт" → "Apple"
  3. Фон: нажмите на иконку сцены → выберите красивый фон (например, "Blue Sky")

Настройте размеры:

  • Корзинка: размер 60% (вкладка "Костюмы" → уменьшить)
  • Яблоко: размер 40%
  • Расположите корзинку внизу экрана (y = -140)
3

Управление корзинкой

Выберите спрайт корзинки и добавьте код:

когда щёлкнут по зёленому флагу
перейти в x: (0) y: (-140)  // Начальная позиция
установить размер (60) %      // Размер корзинки

повтори всегда
  если нажата клавиша [стрелка вправо v] тогда
    изменить x на (10)        // Двигаем вправо
  конец
  
  если нажата клавиша [стрелка влево v] тогда
    изменить x на (-10)       // Двигаем влево
  конец
  
  // Ограничиваем движение в пределах экрана
  если x позиция > (220) тогда
    установить x в (220)
  конец
  
  если x позиция < (-220) тогда
    установить x в (-220)
  конец
конец

Что делает этот код:

  • Корзинка появляется в центре внизу экрана
  • Стрелки влево/вправо двигают корзинку
  • Два условия не дают корзинке выйти за границы экрана
4

Создаем переменные

Нам понадобятся переменные для хранения данных игры:

// Создаем переменные (блок "Переменные")
Очки = 0
Жизни = 3
Скорость = 3
Рекорд = 0

Как создать переменные:

  1. Перейдите в категорию "Переменные"
  2. Нажмите "Создать переменную"
  3. Введите название (например, "Очки")
  4. Повторите для всех переменных

Добавьте код инициализации:

когда щёлкнут по зёленому флагу
установить [Очки v] в (0)
установить [Жизни v] в (3)
установить [Скорость v] в (3)
спрятать переменную [Рекорд v]  // Скрываем рекорд до конца игры
5

Программируем яблоко

Выберите спрайт яблока и добавьте код:

когда щёлкнут по зёленому флагу
спрятаться
установить размер (40) %
установить [цвет v] эффект в (0)

повтори всегда
  показаться
  перейти в x: (выбор случайного от (-200) до (200)) y: (180)
  
  повторять пока не < касается [край v] >
    изменить y на ((0) - (Скорость))
    
    если касается [Basket v] ? тогда
      изменить [Очки v] на (1)
      играть звук [pop v] до конца
      спрятаться
      остановить [этот цикл v]
    конец
    
    если y позиция < (-180) тогда  // Упало мимо
      изменить [Жизни v] на (-1)
      играть звук [laser1 v] до конца
      спрятаться
      остановить [этот цикл v]
    конец
  конец
  
  ждать (выбор случайного от (0.5) до (1.5)) секунд
конец

Объяснение кода:

  1. Яблоко появляется в случайной позиции вверху
  2. Падает вниз с текущей скоростью
  3. Если касается корзинки - добавляем очко и прячем яблоко
  4. Если падает мимо - отнимаем жизнь
  5. Ждем случайное время перед появлением нового яблока
6

Добавляем сложность

Чтобы игра не была слишком простой, добавим увеличение сложности:

// Этот код добавляем к яблоку
когда я получу [увеличить сложность v]  // Создаем сообщение

изменить [Скорость v] на (0.5)
сказать [Сложность увеличена!] на (2) секунд

// Создаем новый спрайт для увеличения сложности (например, часы)
когда щёлкнут по зёленому флагу
ждать (30) секунд
передать [увеличить сложность v] и ждать
ждать (25) секунд
передать [увеличить сложность v] и ждать
ждать (20) секунд
передать [увеличить сложность v] и ждать
// И так далее...

Как создать сообщение:

  1. В категории "События" найдите блок "передать ..."
  2. Нажмите на черный треугольник → "новое сообщение"
  3. Введите "увеличить сложность"
  4. Теперь это сообщение можно отправлять и принимать
7

Конец игры

Добавим проверку окончания игры:

// Этот код можно добавить в отдельный спрайт (например, "Игра")
когда щёлкнут по зёленому флагу
ждать до < (Жизни) < (1) >
остановить [все v]

сказать [Игра окончена!] на (2) секунд
сказать (объединить [Твой счет: ] (Очки)) на (3) секунд

// Проверяем рекорд
если (Очки) > (Рекорд) тогда
  установить [Рекорд v] в (Очки)
  сказать [Новый рекорд!] на (2) секунд
конец

показать переменную [Рекорд v]  // Показываем рекорд после игры
8

Улучшаем игру

Добавим дополнительные элементы для улучшения игры:

// 1. Добавляем разные типы фруктов
// Создайте новые спрайты: банан (дает 2 очка), лимон (отнимает жизнь)

// 2. Добавляем звуки
когда щёлкнут по зёленому флагу
играть звук [Background Music v] до конца  // Фоновая музыка

// 3. Добавляем анимацию при ловле
когда я получу [поймал v]
повторить (10) раз
  изменить размер на (10)
конец
повторить (10) раз
  изменить размер на (-10)
конец

// 4. Добавляем уровни
если (Очки) > (20) тогда
  передать [уровень 2 v] и ждать
  сказать [Уровень 2!] на (2) секунд
конец

Творческое задание

Попробуй улучшить игру следующими способами:

  1. Добавь разные типы фруктов:
    • Золотое яблоко - дает 5 очков (редкое)
    • Гнилое яблоко - отнимает жизнь
    • Банан - дает 2 очка, движется зигзагом
  2. Добавь бонусы:
    • Часы - замедляют время на 10 секунд
    • Сердце - добавляет одну жизнь
    • Молния - ускоряет корзинку на 10 секунд
  3. Сделай уровни:
    • Уровень 1: обычная скорость
    • Уровень 2: два яблока одновременно
    • Уровень 3: яблоки падают под углом
  4. Добавь графические эффекты:
    • Взрыв при ловле яблока
    • Анимация потери жизни
    • Эффект "заморозки" при бонусе

Профессиональный уровень

Создай меню с выбором сложности, таблицу рекордов с именами игроков, систему достижений (поймал 10 яблок, поймал золотое яблоко и т.д.)
