Проект 1: Моя первая анимация

Создай движущегося котика и научись основам Scratch

1

Создаем новый проект

Давайте создадим нашу первую анимацию в Scratch!

  1. Откройте сайт scratch.mit.edu
  2. Нажмите кнопку "Создать" в верхнем меню
  3. Если у вас нет аккаунта, создайте его (это бесплатно!)

Совет

Создание аккаунта позволит сохранять ваши проекты и показывать их друзьям!

2

Знакомство с интерфейсом

В Scratch есть несколько основных областей:

Сцена

Где происходит действие вашего проекта

Спрайты

Персонажи и объекты в вашем проекте

Блоки

Команды для программирования спрайтов

Область кода

Где вы собираете программу из блоков

По умолчанию в проекте уже есть один спрайт - кот Scratch. Мы будем с ним работать!

3

Первые блоки кода

Давайте заставим котика говорить "Привет!"

  1. В палитре блоков выберите категорию "События" (оранжевая)
  2. Перетащите блок когда щёлкнут по зелёному флагу в область кода
  3. Выберите категорию "Внешность" (фиолетовая)
  4. Перетащите блок сказать Привет! на 2 секунды и присоедините его к первому блоку
когда щёлкнут по зелёному флагу
сказать [Привет! Я Котик Scratch!] на (2) секунд

Нажмите на зеленый флаг над сценой, чтобы увидеть результат!

Попробуй сам!

Измени текст в блоке "сказать" и посмотри, что произойдет!

4

Двигаем котика

Теперь научим котика двигаться:

когда щёлкнут по зелёному флагу
повернуться в направлении (90)  // смотрит вправо
идти (10) шагов
сказать [Я иду!] на (1) секунд
повернуться в направлении (-90) // смотрит влево
идти (10) шагов
сказать [И обратно!] на (1) секунд

Что делает эта программа:

  • повернуться в направлении (90) - поворачивает котика направо
  • идти (10) шагов - двигает котика на 10 шагов
  • Обратите внимание: направление 90° = вправо, -90° = влево, 0° = вверх, 180° = вниз
5

Добавляем цикл

Чтобы котик ходил туда-сюда постоянно, добавим цикл:

когда щёлкнут по зелёному флагу
повтори всегда
  повернуться в направлении (90)
  идти (50) шагов
  ждать (0.5) секунд
  повернуться в направлении (-90)
  идти (50) шагов
  ждать (0.5) секунд
конец

Блок повтори всегда заставляет код внутри него повторяться бесконечно!

Важно!

Блок ждать очень важен! Без него котик будет двигаться так быстро, что вы ничего не увидите.

6

Меняем костюмы (анимация)

Котик Scratch имеет несколько костюмов. Давайте создадим анимацию ходьбы:

когда щёлкнут по зёленому флагу
повтори всегда
  следующий костюм
  идти (10) шагов
  ждать (0.2) секунд
конец

Чтобы это работало:

  1. Нажмите на вкладку "Костюмы" над областью кода
  2. Увидите, что у котика есть 2 костюма (поза 1 и поза 2)
  3. Блок следующий костюм будет переключать между ними

Теперь котик не просто движется, а идет как настоящий!

7

Добавляем звуки

Давайте добавим звук шагов:

когда щёлкнут по зёленому флагу
повтори всегда
  следующий костюм
  идти (10) шагов
  играть звук [Pop v] до конца
  ждать (0.2) секунд
конец

Чтобы добавить звук:

  1. Перейдите на вкладку "Звуки"
  2. Нажмите на иконку громкоговорителя в левом нижнем углу
  3. Выберите звук "Pop" из библиотеки
  4. Вернитесь на вкладку "Код"
  5. В категории "Звук" найдите блок играть звук

Творческое задание

Попробуй улучшить свою анимацию:

  1. Добавь фоновую музыку (вкладка "Звуки" → "Выбрать звук" → музыкальные инструменты)
  2. Измени фон сцены (нажми на иконку сцены внизу справа → вкладка "Фоны")
  3. Добавь второго персонажа (нажми на иконку котика внизу справа → выбери спрайт)
  4. Сделай так, чтобы персонажи разговаривали друг с другом

Дополнительная задача

Попробуй сделать так, чтобы котик:

  • Менял цвет при нажатии на него
  • Увеличивался и уменьшался
  • Останавливался при нажатии клавиши пробел
8

Сохраняем проект

Не забудь сохранить свою работу!

  1. Нажми на меню "Файл" вверху
  2. Выбери "Сохранить сейчас"
  3. Придумай название для своего проекта (например, "Моя первая анимация")
  4. Нажми "ОК"

Теперь ты можешь поделиться своим проектом с друзьями!

Поделись проектом

Нажми кнопку "Поделиться" вверху справа, чтобы получить ссылку на свой проект. Отправь ее родителям и друзьям!
