Проект 4: Игра-лабиринт

Создай сложный лабиринт с ловушками и сокровищами

1

Основная идея игры

Мы создадим игру-лабиринт, где игрок должен добраться от старта до финиша, собирая по пути сокровища и избегая ловушек.

Основные элементы:

  • Игрок (управляется стрелками)
  • Стены лабиринта
  • Сокровища (монеты, драгоценности)
  • Ловушки (шипы, враги)
  • Врата на следующий уровень

Механика

Игрок собирает сокровища, набирает очки и пытается пройти несколько уровней лабиринта с увеличивающейся сложностью.

2

Создаем стены лабиринта

Способ 1: Используем спрайты-стены

// Код для игрока
когда щёлкнут по зёленому флагу
перейти в x: (-180) y: (0)  // Стартовая позиция
установить размер (30) %

повтори всегда
  если нажата клавиша [стрелка вверх v] тогда
    изменить y на (5)
    если <касается цвета [#000000] ?> тогда  // Черный цвет - стены
      изменить y на (-5)  // Откатываем движение
    конец
  конец
  
  // Аналогично для других направлений
конец

Как создать стены:

  1. Нарисуйте черные стены прямо на фоне (редактор фонов)
  2. Или создайте отдельный спрайт "Стена" черного цвета
  3. Для сложных лабиринтов лучше рисовать стены на фоне
3

Добавляем сокровища

Создаем монеты, которые игрок должен собрать:

// Код для монеты
когда щёлкнут по зёленому флагу
показаться
установить размер (20) %
перейти в x: (0) y: (100)  // Позиция монеты

// Код для игрока (дополнение)
повтори всегда
  если <касается [Монета v] ?> тогда
    передать [собрал монету v] и ждать
    играть звук [Coin v] до конца
    спрятаться
    ждать (5) секунд
    показаться
  конец
конец

// Код для подсчета очков
когда щёлкнут по зёленому флагу
установить [Очки v] в (0)

когда я получу [собрал монету v]
изменить [Очки v] на (10)

Создайте несколько монет в разных местах лабиринта!

5

Создаем несколько уровней

Добавим возможность проходить несколько уровней:

// Создаем переменные
Уровень = 1
Всего_уровней = 3

// Код для врат на следующий уровень
когда щёлкнут по зёленому флагу
показаться
перейти в x: (180) y: (0)  // Выход в правой части

когда я получу [начать уровень v]
показаться

// Код для игрока (когда достиг выхода)
повтори всегда
  если <касается [Врата v] ?> тогда
    если (Уровень) < (Всего_уровней) тогда
      изменить [Уровень v] на (1)
      передать [следующий уровень v] и ждать
    иначе
      передать [победа v] и ждать
    конец
  конец
конец

// Код для переключения уровней
когда я получу [следующий уровень v]
переключить фон на (объединить [Уровень ] (Уровень))
установить [Очки v] в ((Очки) + (100))  // Бонус за уровень
перейти в x: (-180) y: (0)  // Возвращаем к началу
сказать (объединить [Уровень ] (объединить [ пройден!] (объединить [ Бонус: 100 очков] ( )))) на (3) секунд
6

Добавляем таймер и жизни

Сделаем игру более напряженной:

// Создаем переменные
Время = 60
Жизни = 3
Рекордное_время = 999

// Таймер
когда щёлкнут по зёленому флагу
установить [Время v] в (60)

повтори до < (Время) < (1) >
  ждать (1) секунд
  изменить [Время v] на (-1)
  если (Время) < (10) тогда
    играть звук [Beep v] до конца
  конец
конец

передать [время вышло v] и ждать

// Обработка окончания времени
когда я получу [время вышло v]
остановить [все v]
сказать [Время вышло!] на (2) секунд
сказать (объединить [Твой счет: ] (Очки)) на (2) секунд

// Проверка жизней
повтори всегда
  если (Жизни) < (1) тогда
    передать [поражение v] и ждать
  конец
конец

Творческое задание

Создай продвинутую версию лабиринта:

  1. Добавь разные типы сокровищ:
    • Золото - 10 очков
    • Драгоценные камни - 50 очков
    • Волшебные артефакты - 100 очков + особые способности
  2. Создай разные типы врагов:
    • Патрулирующие - ходят по маршруту
    • Преследующие - следуют за игроком
    • Стражники - стоят на месте, но имеют большую зону видимости
  3. Добавь особые предметы:
    • Ключи - открывают запертые двери
    • Зелья скорости - временно ускоряют игрока
    • Плащи-невидимки - делают игрока невидимым для врагов
    • Карты - показывают весь лабиринт на 10 секунд
  4. Создай систему сохранения:
    • Сохранение прогресса между уровнями
    • Таблица рекордов с именами игроков
    • Достижения (пройти уровень без потерь, собрать все сокровища и т.д.)
